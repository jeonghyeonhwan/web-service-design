# Todo REST API (Spring Boot)

간단한 Todo 관리 기능을 제공하는 Spring Boot 기반의 REST API 프로젝트입니다.  
과제 요구사항인 **POST / GET / PUT / DELETE 각각 2개씩 총 8개의 API 구현**,  
**표준화된 응답 형식**, **미들웨어(로그 필터)** 를 모두 충족합니다.

---

## 📌 기술 스택

- Java 17+
- Spring Boot 3.5.6
- Spring Web
- Validation
- Lombok (선택)
- Gradle
